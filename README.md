# Python-Practicing-2

Algoritmos de classificação - exercícios

01- Ordenação Simples
Dado um vetor de inteiros, implemente o algoritmo Bubble Sort.
Mostre o vetor após cada iteração completa.

02- Selection Sort na Prática
Implemente o Selection Sort para ordenar um array em ordem decrescente.
Conte quantas trocas (swaps) foram realizadas.

03- Insertion Sort com Strings
Dado um array de palavras, ordene-as usando Insertion Sort em ordem alfabética.
Ignore maiúsculas e minúsculas.

04- Comparando Algoritmos
Gere um vetor com 100 números aleatórios.
Ordene usando Bubble Sort e Insertion Sort.
Compare o número de operações realizadas por cada um.

05- Merge Sort Básico
Implemente o Merge Sort.
Mostre como o vetor é dividido até chegar em subvetores unitários.

06- Quick Sort Simples
Implemente o Quick Sort escolhendo o pivô como o último elemento.
Mostre os passos de partição.

07- Ranking de Jogadores
Você recebe uma lista de jogadores com nome e pontuação.
Ordene os jogadores por pontuação (decrescente).
Em caso de empate, ordene alfabeticamente pelo nome.
Restrição: não usar funções prontas de ordenação.

08- Logs de Sistema
Um sistema gera logs com timestamp no formato "YYYY-MM-DD HH:MM:SS".
Ordene milhares de logs de forma eficiente.
Explique qual algoritmo você escolheu e por quê (complexidade importa).

09- Quase Ordenado
Um array está quase ordenado: cada elemento está no máximo 3 posições distante da posição correta.
Desenvolva um algoritmo mais eficiente que O(n log n).
Dica: pense em estruturas auxiliares.
Top K Frequentes

10- Dado um array com milhões de números, encontre os k elementos mais frequentes.
O resultado deve estar ordenado por frequência.
Restrição: otimizar para tempo e memória (evite ordenar tudo)
